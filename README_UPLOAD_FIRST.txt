UPLOAD INSTRUCTIONS

Do NOT upload this ZIP file itself to GitHub.

1. Unzip this file first.
2. Open the unzipped folder.
3. Upload all files and folders inside it to the root of your GitHub repo.

Your repo root must show:
- app
- build.gradle
- settings.gradle
- .github/workflows/android.yml

Then go to GitHub > Actions > Android CI > Run workflow.
The APK will appear in Artifacts as NCP-Mechanical-APK.
