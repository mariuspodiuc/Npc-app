# NCP Mechanical Pro

Flutter Android app MVP for NCP Mechanical.

## GitHub build
1. Upload all files from this folder to the root of your GitHub repo.
2. Open Actions.
3. Run Android CI.
4. Download `ncp-mechanical-pro-debug-apk`.
5. Install `app-debug.apk` on Android.

The workflow creates Android Gradle files automatically, then builds the APK.
